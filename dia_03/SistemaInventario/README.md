## ✏️ EJERCICIO 1: Sistema de Inventario

**Objetivo:** Crear un sistema de inventario usando arrays

**Requerimientos:**

1. **Crear arrays paralelos para:**
   - Nombres de productos (String[]) - 5 productos
   - Precios (double[])
   - Cantidades en stock (int[])
   - Disponibilidad (boolean[])

2. **Funcionalidades:**
   - Mostrar inventario completo
   - Calcular valor total del inventario (precio * cantidad de cada producto)
   - Encontrar producto más caro y más barato
   - Contar cuántos productos tienen stock bajo (menos de 5 unidades)
   - Buscar un producto por nombre
   - Calcular precio promedio de productos
